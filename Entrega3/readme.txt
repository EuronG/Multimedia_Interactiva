Para usar la aplicación sin un control compatible SELECCIONA EXIT THE GAME al inicio cuando se te pida seleccionar un dispositivo

Con el analogo izquierdo o el mouse puedes mover la flecha para dirigir las particulas, mientras mas larga la flecha con más fuerza serán disparadas

Con el analógo derecho puedes mover las particulas luego de haber sido lanzadas

Con el gatillo izquierdo o el primer slider del GUI puedes cambiar el color de las particulas a ser lanzadas

Con el segundo slider se puede cambiar el tamaño de las particulas

Con los numeros 1-7 del teclado cambia el tipo de pincel

Al hacer click derecho en alguna parte del lienzo, se lanzan particulas hacia todas las direcciones desde ese punto y reproduce un sonido
